- **Ubuntu Linux:** Sistema operativo utilizado para configurar los nodos del clúster.
- **Apache Spark:** Motor principal encargado del procesamiento distribuido de datos.
- **Java JDK:** Entorno necesario para la ejecución de Spark.
- **Scala:** Lenguaje utilizado para desarrollar aplicaciones con Spark.
- **Apache Zeppelin:** Entorno interactivo para ejecutar código y visualizar resultados.
- **SSH:** Herramienta utilizada para la comunicación entre nodos.
- **VMware / VirtualBox:** Plataforma de virtualización para crear los nodos del clúster.

> Para evitar problemas de compatibilidad, se recomienda que todos los nodos del entorno de trabajo utilicen las mismas versiones de software y configuración, garantizando un funcionamiento estable del clúster distribuido.
